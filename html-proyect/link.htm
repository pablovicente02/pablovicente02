TYPES OF LINKS IN HTML

In HTML we have to types of links, the relative links and the absolut links, we use the relative links to navigate to our local content, and on the other hand we have the absolut links, this allows to navigate on the Internet, we reffer the user to another Internet Web Page

Relative Links

The structure element of a link is like this:
<a href="index.htm"></a>
This refers to a local page, and its important to know that you can have the page in some folders, so be careful puting the patch in this part, make sure you know were is exactly the page you want to refer.

Absolut Links

The structure of an absolut link is the same, but in te patch we have to enter the URL of the page we want to refer. The bad things in this type of links, is that if the other page close, you dont control the hosting part of the other page, otherwise on the relative links, you can manage more easilly the content of the web page.
